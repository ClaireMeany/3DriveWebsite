3Drive

Features

-Guests may view .obj files from their device in the viewer
-Ability to register an account
-Log on to created account from any device
-Upload .obj files stored to personal account
-Download .obj files stored on personal account
-Delete .obj files from account
-Check uploaded .obj files for non-manifold edges
-View .obj files in a 3D viewer
-Viewer allows for rotating, scaling, and retexturing of the model
-User can organize files into created folders
-Upload .png thumbnails for .obj files, .png files must have the same name as the .obj file to be displayed
-Thumbnail images can be taken directly from the viewer using the camera
